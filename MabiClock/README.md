# MabiClock

## 1. 概要

当アプリケーションはオンラインゲームマビノギのゲーム内時間をマビノギ外においても参照する事を目的としたアプリケーションです。

## 2. 動作環境

Windows 7 Service Pack 1以降がインストールされたWindowsコンピューター。

## 3. 利用方法

3-1. MabiClock.exeをダブルクリック等で起動します。

3-2. 画面に白い文字で時計が表示されます。

3-3. 時計の文字にマウスカーソルを合わせると時計の背景が黒くなります。

3-4. ダブルクリックすると設定が表示され、閉じるボタンで閉じると設定を反映、終了ボタンを押下すると設定を反映してアプリケーションを終了します。

## 4. マビノギとの関係について

当アプリケーションはマビノギのクライアント、並びにマビノギサーバーの解析や通信は行っておりません。
時刻は当アプリケーションを実行しているコンピューターの時刻から取得ています。
